# LEGO Brick Dimensions Reference

Source: [레고 브릭 사이즈 정리 (LEGO Brick Dimensions)](https://vana.kr/?p=267)
URL: https://vana.kr/?p=267

## 단위 (Units)
- 별도 표기 없으면 **mm**
- 실제 설계 기준은 인치: 7.8mm = 5/16″, 9.6mm = 6/16″, 4.8mm = 3/16″

## 클래식 브릭 (Classic Brick)

| 항목 | 크기 |
|------|------|
| 너비 (width) | 7.8 mm |
| 높이 (height, stud 포함) | 9.6 mm |
| 스터드 구멍 직경 | 4.8 mm |
| 판(Plate) 높이 | 3.2 mm (브릭 높이의 1/3) |

> 4개 연속 너비 = 31.8 mm (이론상 32.0이 아닌 이유: 벽 두께 포함 시 약간 다름)

## 테크닉 브릭 (Technic Brick)
- 클래식 브릭과 기본 사이즈 동일
- 측면에 핀(Pin) / 축(Axle) 홀 추가
